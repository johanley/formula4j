Simple demo GUI app, used to evaluate formulas with Formula4j.

This app is much like a simple console. There is a single line at the top for the input of an expression, and an area below for displaying the result of evaluating the expression with formula4j.

This app has these purposes:

  • demonstrate use of formula4j's API
  • quickly experiment with expressions
  • evaluate its timing statistics

This app is not meant to be a general-purpose GUI component.
